This is a readme file for paxdelay analysis created by Xu Jiao

1. the default database in all the java file is paxdelay, change if needed
2. the default analysis year is 2007, change if needed
3. the default sql mode is set to be '', and innodb_buffer_pool_size is 2GB, innodb_file_per_table is 1, please do NOT change
4. Before running each sql file, please specify the database
5. Other specific notification is in the comments of every file
